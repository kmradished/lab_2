# Лабораторная работа 2: Файловая оболочка на Python

## Цель работы
Разработать мини оболочку с файловыми командами.

## Используемые технологии
- **Python модули**: os, datetime, shutil, unittest, tempfile
- **Кодировка**: UTF-8 для поддержки кириллицы

### Основные команды оболочки:

#### Навигация и просмотр
- **`ls [путь]`** - отображение содержимого указанной или текущей директории
- **`cd [путь]`** - смена рабочего каталога
  - `cd ..` - переход в родительский каталог
  - `cd ~` - переход в домашнюю директорию
- **`cat <файл>`** - вывод содержимого текстового файла с поддержкой кириллицы

#### Операции с файлами
- **`cp <источник> <цель>`** - копирование файлов и каталогов
- **`mv <источник> <цель>`** - перемещение или переименование объектов
- **`rm <объект>`** - удаление файлов и каталогов с защитой от случайного удаления системных папок

## Особенности реализации

### Безопасность и обработка ошибок
- Защита от удаления корневой (`/`) и родительской (`..`) директорий
- Подтверждение операции при рекурсивном удалении каталогов
- Комплексная обработка исключений:
  - `FileNotFoundError` - отсутствие файла или каталога
